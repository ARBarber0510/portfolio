Picniq App

This application is a dynamic platform, designed to provide the nearest parks and restaurants based on the users’ location. The backbone of Picniq’s functionality was constructed with Javascript and jQuery, FourSquare and Geocoder location API's.

Team: Alison Barber, Andres Roldan, Joshua Brickman, Steven Schaab

Workday Scheduler App

This site is a calendar application that allows the user to save their schedule for each hour of the day. This app is run in the browser and uses dynamically updated HTML & CSS through the use of jQuery. The calendar date and time is run with the help of Moment.js.
